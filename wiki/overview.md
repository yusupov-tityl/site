# Project Overview

## Company
**Ай-Титул** (itityl.ru) — AI consulting company specializing in enterprise and government AI solutions.

## Website Purpose
Corporate landing page + multi-page site for services, products, company info, press, contacts.

## Tech Stack
- React 19 + TypeScript + Vite
- Tailwind CSS v4
- React Router DOM
- Lucide React icons
- react-helmet-async for SEO
- Google Fonts: Inter (primary), DM Sans (secondary), Bebas Neue (accent)

## Design Reference
Visual style based on [miquido.com](https://www.miquido.com/):
- Dark theme: `#0F0F14` primary, `#16161D` secondary, `#1C1C26` card
- Brand accent: `#FFE600` (yellow/gold)
- Typography: Inter, 60px/500 for H2, 80px/500 for H1
- Patterns: two-column headers, numbered lists, gap-px grids, arrow-in-circle CTAs

## Hero
Full-screen video background (golden 3D abstract animation) with two-column bottom-aligned layout:
- Left: large headline
- Right: description + CTA

## Key Sections (Homepage)
Hero → About → PainPoints → Capabilities → Products → UseCases → Metrics → AISection → ClientLogos → Process → CTABanner → ContactCTA
