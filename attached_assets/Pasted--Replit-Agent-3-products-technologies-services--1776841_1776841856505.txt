# Промпт для Replit Agent — разводящие страницы Ай-Титул

## Контекст

Сайт **itityl.ru** (Ай-Титул) — корпоративный сайт AI-консалтинговой компании. Стек: **Vite + React + TypeScript + Tailwind v4 + framer-motion + wouter**. Дизайн вдохновлён miquido.com: тёмная тема (`#0A0A0A` фон, `#141414` карточки, `#222` бордеры), жёлтый акцент `#FFE600`, шрифты **Inter** (body) и **Space Grotesk** (заголовки).

Уже существуют переиспользуемые компоненты: `MarqueeRow`, `ContactForm`, `HeroBackdrop` — используй их в новых страницах (MarqueeRow между секциями как визуальный ритм, HeroBackdrop в hero-блоках, ContactForm в финальной CTA-секции каждой страницы).

Задача: создать **три разводящие страницы** со всеми блоками строго по ТЗ ниже. Верстать как miquido: широкие секции, нумерация 01/02/…, двухколоночные заголовки, `gap-px` сетки карточек, мягкие framer-motion hover/appear.

## Правила реализации

1. Общие компоненты: `SectionHeader`, `NumberedGrid`, `CardGrid`, `PillRow`, `FAQAccordion`, `CTASection` — три страницы делят общие блоки.
2. Анимации: framer-motion fade-up на appear (`whileInView`, `viewport={{ once: true }}`), hover на карточках — `y: -4` + жёлтый border.
3. Типографика: H1 — Space Grotesk 64/72 desktop, 40/48 mobile; H2 — 48/56 / 32/40; H3 — 28/36; body — Inter 18/28.
4. Цвета: фон `#0A0A0A`, карточки `#141414`, border `#222`, акцент `#FFE600`.
5. Отступы: `py-32` desktop, `py-20` mobile; `max-w-[1440px] mx-auto px-6`.
6. A11y: семантические `h1..h3`, `aria-expanded` в аккордеоне, focus-visible.
7. Mobile: 1 колонка, MarqueeRow сохраняется, FAQ та же аккордеон-логика.
8. HeroBackdrop — в hero каждой страницы с вариацией тона.
9. ContactForm — в финальной секции каждой страницы.
10. Все роуты детальных страниц создать заглушками, чтобы ссылки не 404-или.
